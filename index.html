Welcome to multivariate modeling!

In this set of five lectures and five practicals, you will learn about EFA in the form of factor analysis, and implemented as a structural equation model (SEM).

You'll learn Structural Equation Modeling using free software OpenMx.

At the end you will be comfortable:

  1. Completing a factor analysis using fact.anal
  2. Determining the number of factors needed for a well-fitting model.
  3. Reasoning about why we are doing a factor analysis, and what the result means
  4. Re-implement factor analysis in SEM using umx
  5. Re-implementing a regression in SEM
  6. Testing theories by comparing models, including models with latent variables
